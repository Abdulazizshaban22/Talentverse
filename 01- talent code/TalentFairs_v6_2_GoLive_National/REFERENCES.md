## مراجع تقنية معتمدة
- CloudFront يتطلّب شهادة ACM في us-east-1 للتوزيع العالمي. 
- VPC Origins لربط ALB/NLB خاصة خلف CloudFront.
- ربط AWS WAF مع CloudFront لحماية الويب/الـAPIs.
- Keycloak OIDC Identity Provider (استيراد/إعداد).
- Nafath كدخول موحّد وطني (SSO).
- OneRoster 1.2 (CSV/REST) لبيانات المدارس.
- Expo EAS للبناء السحابي لتطبيقات الموبايل.
