# PDPL Checklist (KSA) — Schools & Minors
- موافقة ولي الأمر قبل جمع/معالجة بيانات الطالب (عمر < 18).
- الغرض محدد ومعلن (تعلم/تقييم/نمو معرفي).
- تقليل البيانات (Data Minimization) — لا تُجمع بيانات حساسة دون ضرورة.
- حق الوصول والتصحيح والسحب (SAR — Subject Access Requests).
- الاحتفاظ لفترة محدودة وخطة محو (Retention & Erasure).
- نقل عبر الحدود فقط وفق الضوابط والتنظيمات المعمول بها.
- سجلات المعالجة (RoPA) وتقييم أثر خصوصية (PIA) للأغراض الجديدة.
